# 项目记忆

## 用户与产品

- 用户姓名：王书琦。
- App 名称：书琦词光。
- 用途：个人使用，提升河南省专升本英语词汇量。
- 设备：iPhone，Apple ID 是日本区。
- 安装偏好：优先保证能用，不走付费路线；推荐 PWA，通过 Safari 添加到主屏幕。
- 重要约束：不要产生任何需要用户付费的东西。

## 当前产品方向

- 手机优先，页面简洁。
- 离线可用。
- 内置专升本英语高频核心词启动包。
- 允许导入 CSV / TXT 词表，便于后续补充河南专升本资料。
- 自动复习：不认识的词更频繁出现，认识的词间隔拉长。
- 有帮助记忆环节：中文回想英文、显示答案。

## 以后怎么改

- 改 App 版本：同步更新 `app.js` 的 `APP_VERSION` 和 `sw.js` 的 `CACHE_NAME`。
- 改词库：编辑 `words.js`，或在 App 内导入 CSV / TXT。
- 改界面：编辑 `index.html` 和 `styles.css`。
- 改复习算法：编辑 `app.js` 里的 `gradeCurrent()`。
- 改离线缓存文件列表：编辑 `sw.js` 的 `APP_FILES`。

## 版本更新策略

- 每次修改后重新打包整个 `henan-vocab-pwa` 文件夹。
- 如果部署到同一个 HTTPS 地址，iPhone 下次打开会发现新版本。
- 更新不会清空学习进度，因为进度保存在浏览器 `localStorage` 的 `shuqi-vocab-state-v1` 里。
- 如果未来需要迁移数据，新增存储 key 时必须写迁移逻辑。
