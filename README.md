# 书琦词光

王书琦的河南专升本英语词汇个人背诵 PWA。

当前版本：v1.0.0。以后每次改功能或词库，都更新 `app.js` 里的 `APP_VERSION` 和 `sw.js` 里的 `CACHE_NAME`。

## iPhone 安装

1. 把本文件夹部署到 HTTPS 网站。
2. 用 iPhone 的 Safari 打开网址。
3. 点底部分享按钮。
4. 选择“添加到主屏幕”。
5. 打开一次 App 后，核心文件会离线缓存。

注意：iOS 离线缓存要求 HTTPS。局域网 HTTP 地址可以预览页面，但不适合作为最终安装地址。

## 免费原则

这个版本不依赖任何付费服务，不需要 App Store 开发者账号，不需要购买证书。最终安装方式建议使用 Safari 的“添加到主屏幕”。

## 导入词表

在“词库”里点“导入 CSV / TXT”。每行格式：

```csv
word,中文释义,例句,记忆提示
```

例子：

```csv
abandon,放弃；抛弃,Never abandon your plan.,a + bandon：把事情扔到一边
```

## 复习规则

- 点“不认识”：5 分钟后优先回来。
- 点“模糊”：约 1 天后复习。
- 点“认识”：间隔逐步变长。

学习数据保存在当前设备浏览器本地。
